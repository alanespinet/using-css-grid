CSS Grid - Rows

The grid-template-rows property

Rows in CSS Grid can be represented by different properties, being grid-template-rows the most common one. This property will accept one or more values representing:

  • The number of rows of the Grid
  • The height of each one

As it happens with Columns, Row heights in CSS Grid can be represented by absolute values, like pixels and relative values like percentages.

Examples

1- We create a grid with 3 rows, each one of 100px height

<div class="grid-container">
    <div class="row-1"></div>
    <div class="row-2"></div>
    <div class="row-3"></div>
</div>

.grid-container {
    display: grid;
    grid-template-rows: 100px 100px 100px;
}

Where each of the 100px appearances relates to one of the rows:

Since we dont have columns defined, each cell of the Grid takes 100% of the container's width.

2- As we did with columns, we can use auto to "fill the gap" between two other rows. For example, in the next example, the grid container is 500px tall. The first column takes 100px of them; the third one takes 50px of them; and the middle one uses auto to take all the "remaining space":

.grid-container {
    display: grid;
    grid-template-rows: 100px auto 50px;
}

This could have been EASILY a scenario where the row-1 was the header, the row-3 was the footer, and the row-2 was the content of a page. For that matter, we would normally use min-height: 100vh; over the grid container to strecth it across the height of the page.

We can also use repeat and minmax with rows the same way as with columns.

3- We can have rows that are not actually defined in size in our grid. When this happens, these rows "share" the rest of the available grid-container's height. For example, following we have a grid-container of 500px, and 4 rows: the first one has 50px height, the second one has 30px height, and the third and fourth, since they don't have any height declared, they will share the the remaing 420px between them, each one taking 210px:

<div class="grid-container">
    <div class="row-1"></div>
    <div class="row-2"></div>
    <div class="row-3"></div>
    <div class="row-4"></div>
</div>

.grid-container {
    display: grid;
    grid-template-rows: 50px 30px;
}

The grid-auto-rows property

4- These rows (like row-3 and row-4 in our example) that are NOT declared with grid-template-rows are called auto tracks and we can give a height to ALL of them through the grid-auto-rows property. In the next code, we assign 100px to them. Look at how we still have, at the end, available space inside the grid container:

<div class="grid-container">
    <div class="row-1"></div>
    <div class="row-2"></div>
    <div class="row-3"></div>
    <div class="row-4"></div>
</div>

.grid-container {
    display: grid;
    grid-template-rows: 50px 30px;
    grid-auto-rows: 100px;
}

NOTE: We also have the grid-auto-columns property, and it has THE SAME effect with columns. It is not, however, as used as grid-auto-rows.
